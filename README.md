# Sweet Suite - צד לקוח (Frontend)

## 🌐 סקירה כללית
הצד של הלקוח בפרויקט **Sweet Suite** נבנה ב-**React** תוך שימוש ב-**Redux** לניהול מצב, ו-**CSS3 Material-UI** לעיצוב. הפרויקט מבוסס על **RESTful API** לתקשורת עם הצד של השרת.

---

### 🛠️ טכנולוגיות וכלים:
- **React**
- **CSS3**
- **Redux-Toolkit**
- **Material-UI**
- **Axios**
- **JWT**
- **React Router**

---

   - **רישום והתחברות** 
   - **בדיקת ולידציות** 
   - **ניהול פרופיל**
   - **סינון מתקדם**
   - **בדיקת תאריך תקין**
   - **כתיבת ביקורות**
   - **צפייה בביקורות**
   - **ניווט**

   - **שיבוץ תאריכים**
   - **תשלום**
